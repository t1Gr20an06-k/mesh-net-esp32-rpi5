# Полевые тесты

## Цели тестирования

1. Реальная дальность LoRa в горном рельефе (прямая видимость vs. склон)
2. Надёжность доставки SOS через 2 ретранслятора
3. Работа captive portal на iOS/Android без доп. настроек
4. Корректность GPS-координат на ESP32

---

## Протокол тестирования

### Тест дальности (FT-01)

**Оборудование:** 2 × RPi5 + SX1262, ноутбук с логгером

**Процедура:**
1. Фиксированная точка (инфо-точка / база) — на возвышении
2. Мобильная точка — удаляется по маршруту с шагом 500 м
3. На каждой точке отправить 10 пакетов PING, записать RSSI/SNR/доставку
4. Зафиксировать GPS-координаты каждой точки тестирования

**Таблица результатов:**

| Точка | Расстояние | Высота разница | RSSI avg | SNR avg | Доставлено / 10 |
|-------|-----------|----------------|----------|---------|-----------------|
| T1 | 500 м | +50 м | | | |
| T2 | 1000 м | +100 м | | | |
| T3 | 2000 м | +200 м | | | |
| T4 | 3000 м | — | | | |
| T5 | 5000 м | — | | | |

**Ожидаемый результат:** RSSI > −130 дБм, доставка > 8/10 на дистанции до 3 км.

---

### Тест SOS через ретрансляторы (FT-02)

**Схема:** Терминал → Инфо-точка → База спасателей

**Процедура:**
1. Все 3 узла вне прямой видимости друг друга
2. Нажать кнопку SOS на терминале
3. Зафиксировать время: T_send → T_receive (на базе)
4. Повторить 20 раз

**Метрики:**
- Процент доставки (цель: 100%)
- Время доставки P50, P95 (цель: < 5 сек)
- Количество дубликатов (цель: не более 3 копий = ровно SOS_REPEAT)

---

### Тест автономности терминала (FT-03)

**Оборудование:** ESP32 + SX1262 + GPS(с телефона) + 18650 3500 мАч

**Режим:** PING каждые 60 с, дисплей включён, Wi-Fi AP активен

**Измерения:** Ток потребления (мА), время работы до разряда

**Ожидаемый результат:** > 24 часов

---

### Тест captive portal (FT-04)

**Устройства для проверки:**
- [ ] iPhone (iOS 16+) — автоматическое открытие портала
- [ ] Android (12+) — автоматическое открытие
- [ ] Android (9–11)
- [ ] Старый Android (без авт. открытия) — ручной переход

**Проверяемое:**
- Портал открывается без дополнительных действий
- Офлайн-карта загружается (тайлы есть локально)
- Кнопка SOS отправляет запрос на relay-node

## Известные ограничения

- LoRa SF10/BW125 даёт ~980 бит/с — достаточно для координат и коротких сообщений, но не для передачи изображений
- В глубоких ущельях дальность значительно снижается
- Captive portal не открывается автоматически при использовании VPN на смартфоне
